---------- Как запустить? ----------
1. Заходишь под своим аккаунтом на страницу https://myaccount.google.com
2. Переходим в меню Security (Безопастность)
3. Если не подключена двух-факторная аутентификация, то подключаем ее. (просто привяжи номер телефона)
4. После подключения двух-факторной аутентификации, на этой же странице, заходим в App Password (на русском наверное так - пароли приложений)
5. Создаем новое приложение (наименование произвольное), после этого гугл выдаст ключ, копируем его и вставляем в appSettings.json в поле "SmtpPassword".
6. В файле appSettings.json прописываем свой почтовый ящик в поле "SmtpUsername".
7. В файле appSettings.json прописываем папку в почте из которой требуется производить парсинг в поле "FolderName" (Опционально, можно оставить значение по умолчанию).
8. Сохраняем appSettings.json
9. Запускаем MailParser.exe
10. Смотрим в консоль
11. В случае успешного парсинга, в этой же папке с программой создаться файл Report.xlsx
